The button is part of the [top navigation]([../components/top-navigation) menu.

![The top navigation menu. The NHS logo is on the left and a button with the text app help is on the right.](/assets/images/top-navigation-2026.png)

## How to use

When a user selects the app help button, a browser overlay shows them help content hosted on the NHS website.

The web page that opens depends on which NHS App screen the user is on. It should usually either be:

- the main [Help with using the NHS App](https://www.nhs.uk/nhs-app/help/) hub
- a sub-topic hub, such as [Help with prescriptions in the NHS App](https://www.nhs.uk/nhs-app/help/prescriptions/) or [Help with appointments in the NHS App](https://www.nhs.uk/nhs-app/help/appointments/)

This lets users access various help articles that may be relevant to them.

## How not to use

It’s best not to route the app help button to a particular article page. Users may need help with something other than the screen they are on in the app. Showing them a hub or sub-topic hub instead gives them more options.

## Research

“App help” was well understood as a label. The button as a whole performed better in usability testing, and was better understood by participants, compared to an older version. The older version used in the app was a question mark icon without a text label.

The button has been tested with visually impaired users and was found to be accessible for screen readers.
